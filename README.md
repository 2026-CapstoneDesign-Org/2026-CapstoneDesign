## **Naming Convention**

|**대상**|**규칙**|**예시**|
|---|---|---|
|클래스|PascalCase|UserService, OrderController|
|메서드/변수|camelCase|getUserById(), userName|
|상수|UPPER_SNAKE_CASE|MAX_RETRY_COUNT|
|패키지|lowercase|com.example.capstone|
|컬럼|snake_case|created_at, user_name|
|테스트 클래스|원본클래스명 + Test|UserServiceTest|
